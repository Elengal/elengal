## Философия

**Elengal** — это не просто нейросеть. Это **платформа для симуляции смыслов**.

```
Мы НЕ создаём токены с параметрами
Мы создаём ПОЛЕ с условиями
Токены появляются, взаимодействуют, СТАНОВЯТСЯ
```

В отличие от традиционных трансформеров, где токены — это статические векторы, в Elengal каждый токен — это **физическая сущность** с:

- **Массой** — инерция смысла, семантическая значимость
- **Энергией** — активность, "температура" токена
- **Фазой** — позиция в многомерном семантическом пространстве
- **Магнетизмом** — полярность отношений с другими токенами
- **Геномом** — ДНК эволюции, способность меняться

## 5 Фундаментальных Сил

| Сила | Физика | Смысл |
|------|--------|-------|
| **Гравитация** | Притяжение масс | Токены с "тяжёлыми" смыслами притягивают другие |
| **Масса** | Инерция | Стабильность смысла, устойчивость к изменениям |

| **Геном** | ДНК | Врождённая способность эволюционировать |

## Математика Мироздания

Elengal построен на **q-анализе** — математике, которую развивал гениальный индийский математик **Сриниваса Рамануджан**.

### Формулы:

```
q-экспонента:
  exp_q(x) = Σ xⁿ / (q; q)ₙ

Символ Похгаммера:
  (q; q)ₙ = П(1 - qᵏ), k=1..n

Тета-функция Якоби:
  θ(z, τ) = Σ(-1)ⁿ q^(n(n-1)) sin((2n-1)πz)

Поле Эйзенштейна:
  ω = e^(2πi/order)
```

При **q → 1** эти формулы превращаются в классическую математику. При **q < 1** — создают уникальную нелинейную динамику, невозможную для обычных трансформеров.

## Быстрый старт

```python
from elengal_v1 import ElengalV1, ElengalConfig

# Создаём конфигурацию
config = ElengalConfig(
    dim=64,              # Размерность пространства
    n_layers=6,          # Слои эволюции
    q_base=0.5,          # Базовое q
    gravity_constant=1.0, # Сила гравитации
    magnetic_constant=0.5, # Сила магнетизма
)

# Создаём модель
model = ElengalV1(vocab_size=10000, config=config)

# Прямой проход
logits, state, attention, q_avg = model(input_tokens)

# Физическое состояние токенов
summary = state.get_physical_summary()
print(f"Масса: {summary['mass_mean']}")
print(f"q-параметр: {summary['q_param_mean']}")
```
