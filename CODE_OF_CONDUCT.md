# Code of Conduct

## Наше обязательство

Мы стремимся создать открытое и инклюзивное сообщество вокруг Elengal. Участие в проекте означает согласие с этим кодексом поведения.

## Наши стандарты

### Позитивное поведение:
- Быть уважительным к разным точкам зрения
- Конструктивно принимать критику
- Фокусироваться на том, что лучше для сообщества
- Проявлять эмпатию к другим участникам
- Помогать новичкам

### Неприемлемое поведение:
- Оскорбления или дискриминация
- Публичное или личное преследование
- Публикация личной информации без согласия
- Любое поведение, которое может считаться неуместным

## Область действия

Этот кодекс поведения применяется:
- В репозитории GitHub
- В Issues и Pull Requests
- В Discussions
- В любом публичном представлении проекта

## Принудительные меры

Нарушения могут привести к:
- Предупреждению
- Временному бану
- Постоянному бану
